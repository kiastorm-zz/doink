## Button Internal

Simple series of buttons for navigation, uses Gatsby `<Link>` element internally and an `<a>` for external. Uses [variants](https://theme-ui.com/guides/variants) from theme-ui to control appearance and styles.

```
import { Button } from "gatsby-theme-kormsen"
<Button to="/page-2" variant="primary">I'm a button</Button>
```
